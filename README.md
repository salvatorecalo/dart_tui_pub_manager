**Dart TUI Pub Manager** is a powerful Terminal User Interface (TUI) designed to make managing Dart and Flutter dependencies a fast, intuitive, and error-free experience. Stop memorizing package names or switching to your browser every time you need a library.

---

## ✨ Key Features

* 🔍 **Interactive Search**: Explore pub.dev in real-time directly from your terminal.
* 📦 **Smart Installation**: Add dependencies to your project with a single keystroke, eliminating typos.
* 🗑️ **Management Center**: View installed packages and remove obsolete ones with instant visual feedback.
* 🛠️ **Developer Friendly**: Built with a robust integration test suite to ensure safe and stable operations on your `pubspec.yaml`.

## 🛠️ Installation

Install **Dart TUI Pub Manager** globally to use it in any Dart or Flutter project:

```bash
dart pub global activate dart_tui_pub_manager
